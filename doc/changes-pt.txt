Google Bookmarks para Android!

* 100% Livre de anúncios e de código livre!
* Suporte para 30 línguas
* Suporta autenticação em duas etapas do Google
* Seguro: Você nunca digita seu username e password dentro do GMarks

Mais funcionalidades:
* Sincronizar/Backup com Google Bookmarks
* Live folder para qualquer etiqueta de favoritos
* Sincronização em segundo plano
* Busca por texto
* Adicionar rapidamente um Favorito a partir da opção Compartilhar no navegador
* Adicionar/atualizar/eliminar favoritos partir do telefone
* Pressione e segure para ver/editar bookmark
* Agora suporta completamente 2.1/Eclair

Chamada de Tradutores! Se você quer ajudar a traduzir o GMarks para seu idioma, veja a página do projeto.

Se você encontrar FC, por favor 'Reporte' para que seja corrigido o problema. Você pode também enviar solicitações de funcionalidades na página do projeto.

Mais detalhes:

Sincronize automaticamente o Google Bookmarks com seu telefone! Efetua sincronizações em segundo plano, o que significa que você estará sempre atualizado, mas só sincroniza o que foi alterado. Use o GMarks para Firefox, Google bookmarklet ou a Barra de ferramentas do Google para enviar os bookmarks rapidamente para o seu telefone.

Escolha qualquer etiqueta para sincronizar com os favoritos do Navegador do seu Android. Isto significa que você pode fazer backup e exportar seus favoritos através de vários dispositivos.

Live Folder lhe dá rápido acesso a qualquer etiqueta. Use-o para criar uma lista de leitura no seu dispositivo!

Possui integração com a Pesquisa Global, o que significa que você pode encontrar rapidamente um favorito usando o botão Pesquisar do telefone ou o widget Google Search. (Você pode ativar isto nas configurações de pesquisa na "Pesquisa") A pesquisa indexa os títulos, etiquetas, URLs e descrições.  

Crie rapidamente um novo Favorito usando a opção Compartilhar no menu do Navegador.

Seguro: Seu username e password não são enviados pelo aplicativo para o Google, e nunca são lidos ou armazenados no GMarks. Todas as conexões são estabelecidas utilizando HTTPS/SSL para assegurar sua completa privacidade. O GMarks também suporta a autenticação em duas etapas (Desde a versão 0.6.4.)  Você deve ter o Google Authenticator instalado.

Palavras chave: Sincronizar backup importar exportar Google Bookmarks Favoritos
